# LLM Council

A multi-model AI agora for structured debate, critique, and synthesis.
An MCP server for discussing complex topics across multiple LLMs — Claude, GPT, Gemini, Grok, and local models.

## Idea

This project explores a way to let multiple AI models discuss the same complex prompt, question, or idea.

Instead of relying on a single model response, the goal is to experiment with workflows where models such as GPT, Claude, Gemini, Grok, and others can answer, challenge assumptions, expose disagreements, and help synthesize a better final response.

Single-model answers miss perspectives. llm-council lets multiple models discuss, debate, and critique each other on the same question, instead of answering in isolation.

## Possible Direction

The project may evolve into an MCP server or orchestration layer for multi-model discussions.

Potential areas to explore:

- independent answers from multiple models
- cross-model critique
- debate and refinement flows
- synthesis of agreements and disagreements
- storing discussion transcripts
